Requisitos de Software

Requisitos gerais:
    - A plaicação deve contar uma página de apresentaçcao do evento
    - A aplicação deve conter um cadastro para clientes
    - A aplicação deve conter uma página para reseva de ingressos
    - A aplicação deve conter uma página de impressão do QR Code 
    - A aplicação deve conter uma página de Administração
    - A aplicação deve conter uma página de contato com o cliente
    - A aplicação deve conter uma página de FAQ

Casos de uso:
    - O cliente entra no site
    - O cliente vai a página de login e faz o cadastro
    - O cliente é redirecionado a um página para selecionar o ingresso
        - Ver formulário de ingresso
    - O cliente é redirecionado a uma página de pagamento
    - O cliente volta ao site par consultar o estatus da sua reserva
    - O cliente é redirecionado a uma página par gerar o QR Code
    - O cliente faz o download do QR Code (pdf se possível)
    - O cliente apresenta o QR Code no dia da apresentação
    - O cliente entra no estabelecimento após a confirmação do scanner

    - O administrador entra no site
    - O administrador faz o login
    - O administrador viualiza 6 listas de ingressos:
        - vendidos
        - pendentes
        - aprovados
        - impressos
        - scanneados 
        - cancelados
    - O administrador seleciona a lista VENDIDOS e contabiliza todas as solicitações do site
    - O administrador seleciona a lista PENDENTES e contabliza todas as solicitações não confirmadas
    - O administrados seleciona a lista APROVADOS e contabiliza dos as solicitações confirmadas
    - O administrador seleciona a lista IMPRESSOS e contabiliza todos os clentes de imprimiram o QR Code
    - O Administrados seleciona a lista SCANNEADOS e contabiliza todas os QR Codes já verificados
    - O administrador seleciona a lista CANCELADOS e contabiliza todas as solicitações canceladas 

    - FAQ: como se cadastrar?

    - FAQ: como conversar com a organização do evento
    - FAQ: como cancelar um ingresso
    - FAQ: como imprimir o QR Code
    - FAQ: minha compra não està liberada, o que faço?
    - FAQ: o que faço com o QR Code
    - FAQ: Preciso imprimir o imgresso?
    - FAQ: como utilizar o ingresso virtual?

    Formulário de ingresso:
        - Nome do evento
        - Logo da Produtora
        - data
        - Horário
        - local
        - número assento (poltrona)
        - Tipo de ingresso (Vip)
